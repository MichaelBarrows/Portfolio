Grade Calculator

HMTL5 CSS3 JavaScript JQuery Object-Oriented Programming Foundation

Summary

A web based application for calculating module grades using weighted averages, and average year grades based on individual module grades and their credit values whilst also showing the grade classification.

Mobile Version

Problem

Having to manually calculate grades using weightings for modules was somewhat complex and became very tedious. Having to repeat this process continuously became very time consuming, and trying to compute the average for the entire year was confusing as this uses credits rather than percentages, meaning that other grade calculators didn't provide a complete solution for this problem.

This solution automates the calculation of weighted averages for modules and the calculation of module grades to determine the average grade for the year, whilst doing this, the web based system shows the outcomes from each stage of the calculations, such as the grade of an assessment as a weighted percentage of the entire module grade and the weighted module grade in comparison to the total number of credits for example:

The following example shows a module based grade

Assignment 1: 80% Achieved, 20% weighting

Assignment 2: 86% Achieved, 80% weighting

To calculate this manually, the grade would be multiplied by the weighting and divided by 100:

A1: (80 * 20) / 100 = 16

A2: (86 * 80) / 100 = 68.8

Total: 16 + 68.8 = 84.8

Development

The functionality of the website was developed using JavaScript and JQuery, which allowed all calculations to be completed on the client-side of the application, meaning that these calculations were completed with no additional requests to the server and the page was not reloaded.

The website was developed using the Foundation framework, as this framework provided basic style elements such as buttons and the responsive grid. Using this framework simplified development, as the emphasis for this project was the functionality of the website, rather than the design.

Testing

The application was tested by manually carrying out calculations and checking that the results from the application matched the manual calculations.

The rest of the functionality was tested during development to ensure that buttons and links functioned correctly.

Future Development

In future, this application has the potential to be further developed with capabilities such as storage (either on the server using PHP and MySQL or using HTML5 Local Storage), this would improve the user experience of the application as users would not have to re-enter grades everytime they visited the site.

The design could be improved to better use the space available in the desktop versions, as there is a lot of white space. Adding a button to add rows would allow more rows to be added as the user requires.
