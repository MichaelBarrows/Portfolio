Connect 4

Summary

A web based implementation of the popular Connect 4 game, using HTML5 Canvas, EmberJS and implemented as a web app for Android using Apache Cordova.

Try me!

Requirements

Requirements consisted of:

  • Must display the game board
  • Must allow markers to be placed
  • Must detect a winnig position
  • Must run as an app on a mobile device
  • Must implement a computer player
  • Must use a JavaScript Framework
